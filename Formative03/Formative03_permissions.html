Directory Listing

Note that files and folders are color-coded:

White: General file
Blue: Directory
Green: Executable or recognized data file
Magenta (Pink): Graphic file
Cyan (Sky Blue): Symbolic link file

Note that these colors can be changed, and there are more of them.
More information is available by entering the following command in the terminal: man dir_colors

For this Exercise we want to make a Python file located in the 'IoT_Files folder executable.
The command to do so is:
sudo chmod +x {filename}
